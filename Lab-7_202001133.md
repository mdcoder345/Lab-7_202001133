following ranges 1 <= month <= 12, 1 <= day <= 31, 1900 <= year <= 2015.The possible output dates would be
previous date or invalid date. Design the equivalence class test cases?

| ----------------- | ----- | ------

For Month:
    
| Equivalence class | Range | Status
| ----------------- | ----- | ------

| ----------------- | ----- | ------
